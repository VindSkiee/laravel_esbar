## Cara Extract Gambar dari Screenshot/Design

### Method 1: Jika Punya File Figma/PSD

1. Buka file design
2. Select layer gambar yang ingin di-export
3. Klik kanan ‚Üí Export
4. Pilih format PNG
5. Centang "Transparent background"
6. Export dengan nama sesuai kebutuhan

### Method 2: Jika Hanya Punya Screenshot

1. Buka screenshot di Photoshop/GIMP/Photopea
2. Gunakan Magic Wand Tool atau Quick Selection
3. Select gambar yang ingin diambil
4. Copy (Ctrl+C) dan Paste ke new document (Ctrl+V)
5. Delete background layer
6. Save as PNG

### Method 3: Online Tools (Paling Mudah!)

#### Remove Background:

1. Buka https://remove.bg
2. Upload screenshot Anda
3. Download hasilnya (background sudah transparan)
4. Rename sesuai kebutuhan

#### Atau gunakan:

- https://www.photopea.com (Photoshop online gratis)
- https://www.canva.com (untuk edit simple)
- https://pixlr.com (photo editor online)

## üìù Step-by-Step: Extract dari Screenshot Anda

Karena Anda sudah punya screenshot lengkap:

### 1. Extract Ice Cream Utama

```
1. Buka screenshot di Photopea/GIMP
2. Zoom in ke bagian 3 ice cream cones
3. Select area ice cream dengan Rectangular Select
4. Copy ‚Üí Paste to new layer
5. Crop ke ukuran pas
6. Export as "ice-cream.png"
```

### 2. Extract Dekorasi (Strawberry, Mint, dll)

```
1. Gunakan Magic Wand atau Select by Color
2. Select background merah
3. Invert selection (Shift+Ctrl+I)
4. Copy selected object
5. Paste to new document
6. Export as PNG dengan nama sesuai
```

### 3. Quick Method dengan Remove.bg

```
1. Screenshot crop bagian yang ingin diambil
2. Upload ke remove.bg
3. Download hasil (background sudah hilang)
4. Rename file sesuai kebutuhan
5. Letakkan di folder assets/
```

## üí° Tips Pro

1. **Ukuran Lebih Besar Lebih Baik**

   - Mudah untuk resize kecil
   - Susah untuk upscale

2. **Penamaan File**

   - Gunakan lowercase
   - Tanpa spasi (gunakan dash: candy-cane.png)
   - Descriptive name

3. **Optimasi Ukuran**

   - Gunakan https://tinypng.com untuk compress
   - Target: < 500KB per file
   - Maintain transparency

4. **Quality Check**
   - Check transparency edges (harus smooth)
   - No white/grey edges
   - Clear and sharp
